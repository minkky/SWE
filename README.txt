Android Studio의 LoginActivity.java에서 에뮬레이터로 실행시키시면, 처음에 로그인 화면을 볼 수 있습니다.


로그인 하기 위해서는 사용자의 아이디와 비밀번호 해당 칸에 모두 "mgjs"를 입력하시면 됩니다.


입력하시면 메인 화면으로 가실 수 있고, 메인 화면에서 원하시는 메뉴를 사용하시면 됩니다.


오른쪽 상단의 네비게이션 바를 누르시면 아이디인 "mgjs"와 밑에 메뉴가 나오는데, 

아이디와 비밀번호를 변경하시려면, mgjs를 클릭하시면 됩니다. 밑의 다른 메뉴를 누르셔도 해당 기능으로 이동하실 수 있습니다.


참고로 모든 기능은 미국인을 위한 앱으로, 모든 전화번호부, 스케줄과 노트는 영어로만 작성해주세요!


그리고 전화번호부에서 전화번호를 입력시 '-'없이 숫자로만 입력해주세요. (예: 01020606940)






+ Software Design과정에서 cancle 표현의 잘못된 점을 뒤늦게 찾아, 함수들에 있는 해당 표현을 cancel로 정정하였습니다.
+ user story 에서 한글로 작성했던 버튼명은 모두 영어로 대체되었습니다.
